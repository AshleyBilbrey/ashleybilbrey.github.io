Ashley Bilbrey's CowBot

About:

CowBot is a campus information bot for Discord servers. It is written in Node.js with the Discord.js library. CowBot is not officially affiliated with UC Davis.

Current Version: v1.0

Features:

  • Dining Common Menus


  • Current Davis Weather


  • More coming soon!

Commands:

  • !cow help - Shows list of commands
  • !cow menu <common> <meal> - Show today's dining common menu. Example: !cow menu tercero breakfast
  • !cow weather - Show the current weather in Davis.
  • More coming soon!
